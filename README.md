# Auto Lyrics

Welcome to Auto Lyrics!

## Installation

To use Auto Lyrics, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/kldisgood/auto-lyrics
    ```

2. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Run the script:

    ```bash
    python main.py
    ```
